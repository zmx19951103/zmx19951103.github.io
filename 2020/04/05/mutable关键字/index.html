mutable关键字

ZMX's Blog
2020-04-05
c++

前言

mutable是c++中的一个关键字,它的应用场景是在类的const成员函数中.当我们想要在类的const成员函数中修改类的某些数据成员的时候，就会用到这个关键字.

mutable数据成员

用const修饰的成员函数表示该该函数不会修改调用该函数的对象的内部状态,const object 只能调用 const member function, 如果要在const member function里修改对象的数据成员是不被允许的，因为这样会破坏对象的常量性.但是如果必须要修改某些成员，就要用到mutable关键字，它表明修改该数据成员不会破坏对象的常量性.
